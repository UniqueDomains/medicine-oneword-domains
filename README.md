## ⚠️ Methodology and caveats

This set is centered on medicine-related keyword domains, but the naming style varies widely by extension and context. Examples such as medical.team, research.college, health.pics, and disease.sale show that some names read as professional and trust-oriented, while others are more promotional, editorial, or niche. When comparing these domains, start with extension fit: a medically sensitive term usually needs a TLD that supports trust and clarity. Then compare ask price against replacement options, since the median ask is $2,695 across a very broad pool. Be cautious with names that may feel off-category, hard to defend, or awkward for patient, clinic, or research-facing use.

- Prioritize TLDs that match trust-sensitive medical use
- Use the $2,695 median ask as a rough price reference
- Avoid pairings where keyword and extension feel misaligned
- Check spelling, intent clarity, and trademark exposure
